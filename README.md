# MPI Course Catalog Enquiry System

## Project setup

Install Dependencies

```
yarn install
yarn add crc
```

Edit environment variable

```
cp .env.example .env.local
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
